# JUnit Starter Kit (Java)

- Use **JUnit 5** + **Mockito** for unit tests.
- Use **Testcontainers** for integration tests (Postgres, Kafka/SQS emulator, Redis as needed).
- Keep tests deterministic (inject clock, seeded data).

## Suggested deps
- org.junit.jupiter:junit-jupiter
- org.mockito:mockito-core
- org.testcontainers:testcontainers
- org.testcontainers:postgresql
