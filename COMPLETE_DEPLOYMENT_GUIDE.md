-9 <PID>\n\n# Or change Docker port mapping in docker-compose.yml\n```\n\n#### Container Won't Start\n```bash\n# Check logs\ndocker logs <container-id>\n\n# Inspect container\ndocker inspect <container-id>\n\n# Restart Docker daemon\ndocker system restart\n```\n\n#### Database Connection Failed\n```bash\n# Check database service\ndocker-compose logs postgres\n\n# Test connection\ndocker-compose exec postgres psql -U postgres -c \"SELECT 1\"\n\n# Check network\ndocker network inspect swipesavvy_swipesavvy-network\n```\n\n#### Out of Memory\n```bash\n# Check Docker disk usage\ndocker system df\n\n# Clean up unused resources\ndocker system prune -a\n\n# Increase Docker memory limit\n# Docker Desktop Settings ‚Üí Resources ‚Üí Memory\n```\n\n### Debug
