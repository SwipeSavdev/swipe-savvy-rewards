# Viewing and Searching Transaction History

## Accessing Transaction History

### Main Transaction View

1. **Open Transactions Tab**
   - Tap "Transactions" in bottom navigation
   - See all transactions in chronological order
   - Most recent transactions appear first

2. **Transaction List View**
   - Each transaction shows:
     - Merchant name and logo
     - Amount (color-coded: red for debits, green for credits)
     - Date and time
     - Transaction status (completed, pending, failed)
   - Scroll down to load older transactions
   - Pull down to refresh

3. **Transaction Details**
   - Tap any transaction for full details
   - View: merchant info, category, payment method, confirmation number
   - See map location (if available)
   - Access receipt (if uploaded)

## Search and Filter

### Search Functionality

**Search by Text:**
- Tap search bar at top of Transactions screen
- Search by: merchant name, amount, description, location
- Results update in real-time as you type
- Clear search with X button

**Search Tips:**
- Search "$50" to find all transactions for that exact amount
- Search "Starbucks" for all Starbucks purchases
- Search "groceries" to find categorized grocery transactions
- Use date format "Dec 2025" for month-specific search

### Filter Options

**Filter by Date:**
- Tap calendar icon (top right)
- Select preset ranges: Today, This Week, This Month, This Year
- Or choose custom date range with start/end dates
- Up to 7 years of history available

**Filter by Category:**
- Tap filter icon
- Choose categories: Groceries, Dining, Transportation, Shopping, Bills, etc.
- Select multiple categories
- See spending breakdown by category

**Filter by Type:**
- Debit Card Purchases
- ACH Transfers
- Direct Deposits
- ATM Withdrawals
- Peer-to-Peer (P2P) Payments
- Bill Payments
- Refunds

**Filter by Status:**
- Completed (posted)
- Pending (authorized but not posted)
- Failed/Declined
- Refunded
- Disputed

**Filter by Amount:**
- Set min/max amount range
- Find large purchases quickly
- Track small recurring charges

## Exporting Transactions

### Export Methods

1. **Email Export**
   - Tap export icon (↗️) in top right
   - Select date range
   - Choose format: PDF or CSV
   - Transactions sent to registered email
   - Arrives within 5 minutes

2. **Download to Device**
   - Export as CSV for Excel/Sheets
   - Export as PDF for statements
   - Saved to device Downloads folder
   - Can be printed or shared

3. **Tax Documents**
   - Generate year-end statement
   - Includes all transactions by category
   - PDF format optimized for accountants
   - Available each January for previous year

### Export Formats

**CSV (Spreadsheet):**
- Opens in Excel, Google Sheets, Numbers
- Columns: Date, Merchant, Category, Amount, Status, Confirmation
- Easy to pivot, analyze, and create reports

**PDF (Statement):**
- Professional formatted statement
- Includes account summary
- Sorted by date
- Shows running balance
- Suitable for loan applications, proof of income

## Transaction Categories

### Auto-Categorization
SwipeSavvy automatically categorizes transactions:

- **Dining:** Restaurants, cafes, food delivery
- **Groceries:** Supermarkets, convenience stores
- **Transportation:** Gas, tolls, parking, ride-sharing
- **Shopping:** Retail, online purchases, clothing
- **Bills & Utilities:** Internet, phone, electric, water
- **Entertainment:** Movies, concerts, streaming services
- **Health:** Pharmacy, doctor, fitness
- **Travel:** Hotels, flights, car rentals
- **Cash:** ATM withdrawals
- **Transfers:** Between accounts, P2P payments

### Manual Re-categorization
- Tap transaction > Category
- Select correct category from list
- SwipeSavvy learns from your corrections
- Future similar transactions auto-categorize correctly

## Dispute and Report Issues

### Dispute a Transaction

1. **Initiate Dispute**
   - Tap transaction > "Report Problem"
   - Select reason: Fraud, Wrong Amount, Not Authorized, Duplicate
   - Add notes and upload evidence (optional)
   - Submit dispute

2. **Dispute Process**
   - Provisional credit issued within 10 business days (for eligible disputes)
   - Investigation takes 30-90 days
   - Status updates in Notifications
   - Final decision communicated via email

### Report Errors
- Tap "Report Error" on transaction
- Describe issue (wrong merchant, incorrect amount)
- Support team reviews within 24 hours
- Correction applied if validated

## Transaction Insights

### Spending Analytics
- Tap "Insights" tab
- See spending by category (pie chart)
- Monthly spending trends (line graph)
- Compare month-over-month changes
- Top merchants by spend

### Budgeting Integration
- Set budget limits by category
- Track spending against budget
- Alerts when approaching limits
- Rollover unused budget (optional)

## Troubleshooting

### Transaction Missing
- Check date filter (may be outside range)
- Ensure transaction has posted (not still pending)
- Refresh transaction list
- Contact support if truly missing

### Wrong Category
- Tap transaction > Edit Category
- Select correct category
- Mark "Remember for future" to train system

### Can't Export
- Check email address is verified
- Ensure date range is not too large (max 1 year per export)
- Try different format (CSV vs PDF)
- Check spam folder for export email
