# Opt-in Method 1: Mobile App Enrollment

## User Experience Flow

### Step 1: Welcome Screen
**Screen Title:** "Welcome to SwipeSavvy!"

**Visual:** App logo with gradient background, benefit icons displayed in cards

**Exact Verbiage Presented:**
```
Welcome to SwipeSavvy!

Join our rewards program and start earning points on every purchase.

✓ Earn points on every purchase
✓ Exclusive member discounts
✓ Early access to deals via SMS
✓ Birthday rewards & surprises

[Get Started]  [Maybe Later]
```

**User Action:** User taps "Get Started" button to proceed

---

### Step 2: SMS Consent Screen
**Screen Title:** "Stay Connected"

**Visual:** Message icon, phone number input field, consent checkboxes

**Exact Verbiage Presented:**
```
Stay Connected

Get exclusive offers, order updates, and rewards notifications via text.

Phone Number: [_________________]

☑ I agree to receive recurring automated marketing text messages
  at the phone number provided. Consent is not a condition of purchase.

☑ I agree to the Terms of Service and Privacy Policy.

[Confirm & Continue]  [Skip for Now]

Msg & data rates may apply. Msg frequency varies. Reply STOP to cancel,
HELP for help. Carriers are not liable for delayed or undelivered messages.
```

**User Action:**
1. User enters phone number
2. User checks both consent boxes (required)
3. User taps "Confirm & Continue"

---

### Step 3: Confirmation Screen
**Screen Title:** "You're All Set!"

**Visual:** Green checkmark success icon, notification preference toggles

**Exact Verbiage Presented:**
```
You're All Set!

Welcome to SwipeSavvy Rewards! You'll receive a confirmation text shortly.

Your Notification Preferences:
• Promotional offers     [ON]
• Order updates          [ON]
• Rewards alerts         [ON]
• Weekly deals           [ON]

[Start Earning Points]

You can update your preferences anytime in Settings → Notifications
```

**User Action:** User taps "Start Earning Points" to enter main app

---

### Confirmation SMS Sent
**Message Content:**
```
SwipeSavvy: Welcome! You're now enrolled in SwipeSavvy Rewards.
Reply STOP to opt out, HELP for help. Msg&data rates may apply.
```

---

## Compliance Notes
- Consent is obtained via explicit checkbox (not pre-checked)
- Phone number is entered by the user (not pre-filled)
- STOP/HELP instructions provided on consent screen
- Msg & data rates disclosure included
- Terms of Service and Privacy Policy linked
- Consent is not a condition of purchase (clearly stated)
- Confirmation message sent immediately upon opt-in
