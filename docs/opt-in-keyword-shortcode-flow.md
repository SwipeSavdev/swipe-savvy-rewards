# Opt-in Method: Keyword to Short Code (Over the Phone / In-Person)

## User Experience Flow

### Step 1: Customer Sees Signage or Hears Verbal Prompt

#### Option A: In-Store Signage
**Visual:** Counter sign, poster, or digital display

**Exact Verbiage Displayed:**
```
         SwipeSavvy
       Rewards Program

    Join Our Rewards!
  Get exclusive deals & earn points

  ┌─────────────────────────┐
  │   TEXT THIS KEYWORD     │
  │                         │
  │        JOIN             │
  └─────────────────────────┘

  ┌─────────────────────────┐
  │   TO THIS NUMBER        │
  │                         │
  │       12345             │
  └─────────────────────────┘

  ✓ Exclusive member discounts
  ✓ Early access to sales
  ✓ Birthday rewards

─────────────────────────────────────────
By texting JOIN to 12345, you consent to
receive recurring automated marketing
messages from SwipeSavvy. Msg & data rates
may apply. Msg frequency varies. Reply
STOP to cancel, HELP for help.
```

#### Option B: Employee Verbal Script (In-Person / Phone)
**Employee Says:**
```
"Would you like to join our rewards program and receive
exclusive deals via text message? Just text JOIN to 12345
from your phone. You'll get a signup bonus and exclusive
member discounts!"
```

**Required Disclosure (Employee Must Say):**
```
"By texting JOIN, you're agreeing to receive recurring
marketing texts. Message and data rates may apply.
You can text STOP anytime to opt out."
```

**⚠️ IMPORTANT:** Employee must NOT enter customer's phone number for them. Customer must text the keyword themselves to provide valid consent.

---

### Step 2: Customer Texts Keyword to Short Code

**Customer Action:** Opens messaging app and sends text

**Message Sent by Customer:**
```
To: 12345
Message: JOIN
```

---

### Step 3: Automated Confirmation Response

**System Auto-Reply (Immediate):**
```
SwipeSavvy: Welcome! You're now enrolled in our rewards
program. Reply STOP to opt out, HELP for help. Msg&data
rates may apply. Msg frequency varies.
```

## Supported Keywords

| Keyword | Action | Response |
|---------|--------|----------|
| JOIN | Opt-in to rewards | Welcome message + confirmation |
| STOP | Opt-out | Unsubscribe confirmation |
| HELP | Get help | Help message with contact info |
| INFO | Get info | Program details |

## Compliance Notes

### Signage Requirements:
- Clear display of keyword and short code
- Consent language visible on signage
- Msg & data rates disclosure
- STOP/HELP instructions included
- Message frequency disclosure

### Verbal Script Requirements:
- Employee must verbally disclose opt-in terms
- Employee must NOT enter phone number on behalf of customer
- Customer must physically send the text message themselves
- Verbal consent alone is NOT sufficient - customer must text keyword

### Auto-Response Requirements:
- Immediate confirmation sent upon receiving keyword
- Brand name (SwipeSavvy) identified in message
- STOP instructions included
- HELP instructions included
- Msg & data rates disclosure
- Message frequency disclosure

### Record Keeping:
- System logs timestamp of keyword received
- Phone number automatically captured from incoming text
- Opt-in method recorded as "keyword"
- All consent records retained for compliance

---

## Sample Employee Training Points

1. **Always offer the rewards program** at checkout or during calls
2. **Read the full script** including disclosure language
3. **Never text on behalf of the customer** - they must send JOIN themselves
4. **Point to signage** if available for reference
5. **Answer questions** about the program benefits
6. **Confirm they understand** they'll receive marketing texts
