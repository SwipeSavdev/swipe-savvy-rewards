A. Quick Reference Commands

# Daily Developer Commands
nvm use 20.13.0          # Use correct Node.js version
npm ci --workspaces      # Install dependencies
npm run lint -- --fix    # Fix linting issues
npm run type-check       # Check TypeScript
npm run test             # Run tests

# Debugging
npm run test:watch       # Watch mode
npm run test:coverage    # Coverage report
npm run lint --debug     # Lint with debug info

# Deployment
git push origin main     # Triggers auto-deploy
# Monitor: GitHub Actions > Deploy workflow

E. Glossary of Terms

Term Definition
CI/CD Continuous Integration/Continuous Deployment - automated testing and deployment
Workflow GitHub Actions automation that runs on code push or schedule
SLA Service Level Agreement - maximum time to resolve critical issues
Rollback Reverting a bad deployment to previous known-good version
Runbook Step-by-step procedure for handling a specific type of incident
CVSS Common Vulnerability Scoring System - measures severity of vulnerabilities
Version Pinning Locking dependencies to exact versions to ensure reproducibility
